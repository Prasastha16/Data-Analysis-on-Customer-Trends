# Data-Analysis-on-Customer-Trends
End-to-End Retail Customer Shopping Trends Analysis Using SQL, Python, and Power BI

Overview

This project presents an end-to-end Data Analytics workflow focused on analyzing customer shopping behavior in the retail industry. The objective is to transform raw retail transaction data into actionable business insights using Python, SQL, and Power BI. The project covers data cleaning, exploratory data analysis (EDA), SQL-based business queries, dashboard creation, and executive reporting.

The analysis helps identify customer purchasing patterns, sales trends, product performance, and key factors influencing customer behavior, enabling data-driven decision-making.
